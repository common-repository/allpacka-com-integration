Pactic.ro shipping method for WooCommerce

== Description ==

= English =

Pactic offers a cost effective way of delivering your parcels across Hungary, Romania, the EU, USA, and around the world. Trusted by thousands of online sellers.

Link your WooCommerce webshop to [Pactic’s](https://www.pactic.com/) online logistics service. The **Pactic (previously Allpacka.ro)** plugin allows you to automatically place orders. You can transfer the selected orders with a single click to the [Pactic](https://www.pactic.com/) system, which returns an order confirmation number, the Waybill number and shipping label (if you are using a service that requires that the label must be printed).

Functions:

*	Sending order information to [Pactic](https://www.pactic.com/)
*	We store the order number and the number of the waybill for each successful transfer
*	The generated waybill can be downloaded, which can be later viewed and printed
*	Bulk label generation, saving multiple labels in a single PDF file
*	Test environment for sending test orders
*	Package Status Tracking
*	Unique name for AWD downloading
*	Custom setting options for COD


To use this plugin, you must:

*	have a [business contract](https://dashboard.pactic.com/en/business-registration), for this please contact us on the link above or on [sales_hu@pactic.com](mailto:sales_hu@pactic.com)

General Terms and Conditions: [Pactic ÁSZF](https://allpacka.com/terms-conditions-tc/)

= Românesc =

Compania noastră oferă o soluție eficientă de livrare a pachetelor pe plan național, Ungaria, UE, SUA sau oriunde în lume. Alătură-te miilor de clienți mulțumiți!

Conectează-ți magazinul online WooCommerce cu platforma noastră logistică **[Pactic](https://www.pactic.com/ro/home) (anterior Allpacka.ro)**. Comenzile de livrare pot fi transferate în sistemul [Pactic](https://www.pactic.com/ro/home) cu un singur click, care returnează un număr de comandă, numărul de AWB și o versiune printabilă a acestuia.

Funcționalități:

*	Se trimit datele despre comenzile magazinului online către [Pactic](https://pactic.ro)
*	Se stochează Id-ul comenzii [Pactic](https://pactic.ro) și numărul de AWB pentru fiecare comandă procesată cu succes
*	AWB-urile pot fi descărcate, vizualizate și tipărite oricând ulterior
*	Generarea de etichete de transport se poate face bulk și descărca mai multe etichete într-un singur fișier printabil tip .PDF
*	Modulul de testare permite trimiterea comenzilor de test
*	Coletele pot fi urmărite online
*	Se pot utiliza opțiuni pentru setări preferate ale plăților ramburs


Pentru a utiliza acest plugin sunt necesari următorii pași:

*	Se va încheia un [contract comercial](https://dashboard.pactic.com/ro/autentificare) cu [Pactic](https://pactic.ro) prin care se acordă plata la termen
*	Pentru semnarea contractului, te rugăm să ne contactezi prin linkul de mai sus sau la adresa de e-mail [sales_ro@pactic.com](mailto:sales_ro@pactic.com )

Termenii și condițiile generale [Pactic](https://pactic.ro) cu privire la furnizarea serviciilor poștale și curierat pot fi consultate [aici](https://allpacka.com/terms-conditions-tc). 

== Installation ==
= English =
1. Download this plugin.
2. In WordPress, use Plugins / Add New / Upload Plugin to install it.
3. Go to WooCommerce / Settings to set up the plugin.
4. The plugin is ready to use.

= Românesc =
1. Se descarcă acest plugin.
2. În WordPress, se utilizează Plugin / Add New / Upload Plugin pentru a se instala.
3. Se accesează WooCommerce / Settings pentru a configura plugin-ul.
4. Plugin-ul este gata de utilizare.
